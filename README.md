# 🔐 Password Manager

A simple and secure Password Manager built using React and Vite.
This app allows users to store, view, copy, and manage their passwords efficiently.

---

## 🚀 Features

* 🔒 Store website credentials (site, username, password)
* 👁️ Show / Hide password functionality
* 📋 Copy password to clipboard
* 💾 Data stored in localStorage
* ⚡ Fast performance using Vite
* 🎨 Clean UI with Tailwind CSS

---

## 🛠️ Tech Stack

* React
* Vite
* JavaScript (ES6+)
* Tailwind CSS

---

## 📦 Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/your-username/password-manager.git
   ```

2. Navigate to project folder:

   ```bash
   cd password-manager
   ```

3. Install dependencies:

   ```bash
   npm install
   ```

4. Run the app:

   ```bash
   npm run dev
   ```
